argopandas package API¶

The root package contains the interface for argopandas.

>>> import argopandas as argo
argopandas.default_mirror() → argopandas.mirror.Mirror¶

Get the default mirror (e.g., used to populate the global indexes).

>>> import argopandas as argo
>>> argo.default_mirror()
argo.CachedUrlMirror('https://data-argo.ifremer.fr')
argopandas.filename(path: Union[str, Iterator[str]]) → Union[str, Iterator[str]]¶

Download a one or more files and return the filename.

argopandas.nc(path: Union[str, Iterator[str]]) → Union[str, Iterator[argopandas.netcdf.NetCDFWrapper]]¶

Get a netCDF4.Dataset or iterate over multiple data sets.

argopandas.set_default_mirror(mirror: argopandas.mirror.Mirror) → argopandas.mirror.Mirror¶

Set the default mirror.

Parameters

mirror – One of url_mirror() or file_mirror().

argopandas.url(path: Union[str, Iterator[str]]) → Union[str, Iterator[str]]¶

Return the URL to one or more files. If default_mirror() is a file_mirror(), this will be a file:// URL.
